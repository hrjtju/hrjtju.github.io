2 Convexity

2.1 Convex Sets and Convex Functions

Convex Set
A set $ \mathcal{X}$ is called a convex set iff $ \forall (x, y, \psi) \in \mathcal{X}\times \mathcal{X}\times[0,1)$ , $ (1-\psi)x + \psi y \in \mathcal{X}$$

Convex function
A function that always lies below its chords
$ \forall (x, y, \psi) \in \mathcal{X}\times \mathcal{X}\times[0, 1)$ , $ f((1-\psi)x + \psi y) \leqslant (1-\psi)f(x) + \psi f(y)$$

Convex <=> Optimization

  • Convex domain $ \mathcal{X}$$

  • First order: Function $ J: \Omega \rightarrow \mathbb{R}$ is convex iff $ \forall x, y \in \Omega$ , we have $ J(y) \geqslant J(x) + \nabla J(x)^{T}(y-x)$$

3.1 The Black-box Model

With unlimited computatino resources and objective function $ f: \mathcal{X} \rightarrow \mathbb{R}$ , which is unknown but can be accessed through queries to oracles:

  • A zeroth order oracle: $ \mathcal{X} \rightarrow \text{Im}(f), x\mapsto f(x)$$
  • A first order oracle: $ \mathcal{X} \rightarrow \text{Im}(\partial f), x \mapsto g_{x}$$
