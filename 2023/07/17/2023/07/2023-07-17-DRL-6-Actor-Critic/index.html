1.6 MC evaluation with funcion approximation

![[attachments/Pasted image 20230711204317.png|300]]

Instead of applying the aforementioned approach, consider using a neural network parametrized by $\boldsymbol\phi$ that takes in state
$\boldsymbol s{t}$ and estimates the value $\hat{V}^{\pi}$ of the state. Then the evaluation problem has been transformed into an deep
learning task, where the states and reward pairs
$\left{ \left( \boldsymbol s{i,t}, \sum\limits{t’=t}^{T} r(\boldsymbol s{i,t’}, \boldsymbol a{i,t’}) \right) \right}$ is considered
as training data. As the neural network can be treated as a continuous map from state space
$\mathscr{S}=\mathbb{R}^{n{s}}$ into $\mathbb{R}$, nearby states are expected to have similar outputs.

Thus we can train the so-called “critic” network to learn how to judge whether a state is good or not by simply applying supervised regression

2 Actor-Critic

2.1 Actor-critic algorithm

Batch actor-critic algorithm

repeat the following process until convergence:

  1. Sample ${\boldsymbol s{i}, \boldsymbol a{t}}$ from $\pi_{\boldsymbol\theta}(\boldsymbol a|\boldsymbol s)$ on a robot
  2. Fit $V^{\pi}_{\boldsymbol\phi}(\boldsymbol s)$
  3. Evaluate $A^{\pi}(\boldsymbol s{i}, \boldsymbol a{i}) = r(\boldsymbol s{i}, \boldsymbol a{i}) + \hat{V}^{\pi}(\boldsymbol s{i}’) - \hat{V}^{\pi}(\boldsymbol s{i})$
  4. Calculate policy gradient $\nabla_{\boldsymbol\theta} J(\boldsymbol\theta)$
  5. Update policy $\boldsymbol\theta \leftarrow \boldsymbol\theta + \alpha \nabla_{\boldsymbol\theta}J(\boldsymbol\theta)$

2.3 Online actor-critic algorithms with discount

Online actor-critic algorithm with discount

repeat the following process until convergence:

  1. Take action $\boldsymbol a \sim\pi_{\boldsymbol\theta}(\boldsymbol a|\boldsymbol s)$ and get $(\boldsymbol s, \boldsymbol a, \boldsymbol s’, r)$
  2. Update $V^{\pi}{\boldsymbol\phi}(\boldsymbol s)$ using target $r + \hat{V}^{\pi}{\boldsymbol\phi}(\boldsymbol s’)$
  3. Evaluate $A^{\pi}(\boldsymbol s{i}, \boldsymbol a{i}) = r(\boldsymbol s{i}, \boldsymbol a{i}) + \hat{V}^{\pi}(\boldsymbol s{i}’) - \hat{V}^{\pi}(\boldsymbol s{i})$
  4. Calculate policy gradient $\nabla_{\boldsymbol\theta} J(\boldsymbol\theta)$
  5. Update policy $\boldsymbol\theta \leftarrow \boldsymbol\theta + \alpha \nabla_{\boldsymbol\theta}J(\boldsymbol\theta)$

3.3 Fixing the value function

Examine the following algorithm

Broken actor-critic algorithm

repeat the following process until convergence:

  1. Take action $\boldsymbol a \sim\pi_{\boldsymbol\theta}(\boldsymbol a|\boldsymbol s)$ and get $(\boldsymbol s, \boldsymbol a, \boldsymbol s’, r)$ before store in replay buffer $\mathcal{R}$
  2. Sample a batch $(\boldsymbol s, \boldsymbol a, \boldsymbol s’, r)$ from $\mathcal{R}$
  3. —> Update $V^{\pi}{\boldsymbol\phi}(\boldsymbol s)$ using target $r + \hat{V}^{\pi}{\boldsymbol\phi}(\boldsymbol s’)$
  4. Evaluate $A^{\pi}(\boldsymbol s{i}, \boldsymbol a{i}) = r(\boldsymbol s{i}, \boldsymbol a{i}) + \hat{V}^{\pi}(\boldsymbol s{i}’) - \hat{V}^{\pi}(\boldsymbol s{i})$
  5. —> Calculate policy gradient $\nabla{\boldsymbol\theta} J(\boldsymbol\theta)\approx \frac{1}{N}\sum\limits{i}\nabla{\boldsymbol\theta} \log \pi{\boldsymbol\theta}(\boldsymbol a{i}|\boldsymbol s{i})\hat{A}^{\pi}$
  6. Update policy $\boldsymbol\theta \leftarrow \boldsymbol\theta + \alpha \nabla_{\boldsymbol\theta}J(\boldsymbol\theta)$
