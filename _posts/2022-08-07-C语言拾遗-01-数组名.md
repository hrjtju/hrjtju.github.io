```C
#include<iostream>
using namesoace std;

int main()
{
    int a[4] = {1, 3, 5, 7};
    
    cout << a << endl;
    cout << a + 1 << endl;
    cout << &a << endl;
    cout << &a + 1 << endl;
    cout << *(&a) << endl;
    cout << *(&a) + 1 << endl;
}
```



**当数组名不作为`sizeof`、`Alignof`、`&`(取地址符)的操作数时，将自动转换为指向数组首元素的指针**

**`&a`返回一个指向数组的指针，而非指向数组首元素的指针。**



假设数组首元素的地址为`0028F7C4`，则输出依次为

```
0028F7C4
0028F7C8
0028F7D4
0028F7C4
0028F7C8
```
